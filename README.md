# Aplicacion de Trivia, como trabajo practico 



## Tecnologias
* Node.js
* Express
* Express Messages, Session
* MongoDB + Mongoose
* Pug (template)
* Passport (autentiacion)
* BCrypt (encriptado)

### Version
1.0.0



### Installation

Para instalar las dependencias

```sh
$ npm install
```
Para ejecutar la aplicacion

```sh
$ npm start
```
